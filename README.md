# 🛡 Website Blocker Pro

A modern **Python-based website blocking application** designed to improve productivity, digital wellbeing, and focus by controlling access to distracting websites.

Built with **Python + Tkinter**, Website Blocker Pro provides a secure dashboard interface with password protection, website management, and system-level blocking using the operating system's hosts file.

---

## 🚀 Features

### 🔒 Security Features
- Password-protected access
- SHA-256 password hashing
- Secure password change system
- Forgot password recovery
- Administrator/root permission validation

### 🌐 Website Management
- Block selected websites instantly
- Unblock websites whenever needed
- Supports popular platforms:

  - Facebook
  - Instagram
  - TikTok
  - Snapchat
  - LinkedIn
  - Pinterest
  - Reddit
  - YouTube

### 🖥 Modern Dashboard UI

- Dark professional security interface
- Real-time clock display
- System status monitoring
- Quick action controls
- Responsive Tkinter layout

### 📁 System Features

- Automatic hosts-file backup
- Activity logging
- Cross-platform support:
  - Windows
  - Linux
  - macOS
